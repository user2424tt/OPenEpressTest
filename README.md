**Express Test** is a convenient service for *creating and conducting survey*s.

The app allows users to register accounts, which track survey statistics for each user. Authorization is required to create and participate in surveys.

- Use:

	- Have the latest version of Python installed with the Pip package manager;
	- Install all project dependencies with the command(In this case, you need to be in the "server" folder):
	- `pip install -r .\requirements.txt"
	-  Start the server with the command(in the "server" folder):
	- `python main.py`

	A virtual environment is also available. The default HTTP port is *3000*. It can be changed in the *main.py*.
